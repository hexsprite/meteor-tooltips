Tooltips for Meteor

Demonstrating Meteor tooltips, by Lookback.

  • GitHub
  • Atmosphere
  • Source for this demo

Simply embed the tooltips template in your master template/layout and attach tooltips with adding the data-tooltip attribute with the tooltip text.

Default tooltip.

<button data-tooltip="I'm a tooltip!">A tooltip trigger</button>

You can use HTML tags in your tooltips.

<button data-tooltip="You can use <strong>HTML</strong> in here too.">A tooltip trigger</button>

You can use data-tooltip-element to refer to an external element containing markup for the tooltip.

<button data-tooltip-element="#tooltip-el">A tooltip trigger</button>

Tooltip to the left (west). Data attribute values:

  • n - Top (default).
  • s - Bottom.
  • e - Right.
  • w - Left.
<button data-tooltip="I'm a tooltip!" data-tooltip-direction="w">A tooltip trigger</button>





Offset tooltip by specifying the data-tooltip-top and data-tooltip-left attributes. Both attributes take plain numbers (negative and positive), counted in pixels.

The end result of this also depends on data-tooltip-direction. 

<button data-tooltip="I'm a tooltip!" data-tooltip-top="50">A tooltip trigger</button>

<button data-tooltip="I'm a tooltip!" data-tooltip-top="-50">A tooltip trigger</button>

<button data-tooltip="I'm a tooltip!" data-tooltip-direction="w" data-tooltip-top="30" data-tooltip-left="50">A tooltip trigger</button>

<button data-tooltip="I'm a tooltip!" data-tooltip-direction="e" data-tooltip-top="-30" data-tooltip-left="50">A tooltip trigger</button>

<button data-tooltip="I'm a tooltip!" data-tooltip-direction="n" data-tooltip-top="0">A tooltip trigger</button>

<button data-tooltip="I'm a tooltip!" data-tooltip-direction="e" data-tooltip-left="0">A tooltip trigger</button>
You can add custom classes to a tooltip via data-tooltip-classes.

It's possible to completely disable the tooltips, or just for a certain viewport. By setting the Tooltips.disable option (defaults to false), you can pass in true to disable all tooltips, or a matchMedia string which disables all tooltips for that viewport.

# Disable for all
Tooltips.disable = true

# Disable for devices/browsers over 500 px in width
Tooltips.disable = '(min-width: 500px)'

# Disable for devices/browser below 400 px in width
Tooltips.disable = '(max-width: 400px)'

You can also disable individual tooltips directly from the markup, by setting the data-tooltip-disable attribute:

<!-- Disables *this* tooltip for browsers below 400px in width. -->

<button data-tooltip="I'm a tooltip!" data-tooltip-disable="(max-width: 400px)">A tooltip trigger</button>
Nested element

Tooltips should work as expected when attaching them to a parent element. 

<span data-tooltip="Top-level text"><strong>Nested element</strong></span>
{{#tooltip}} {{#if show}} {{/if}} {{/tooltip}}

Tooltips should be removed when the attached element is removed.

Tooltip Triggers







You can set different triggers for tooltips: hover, click, focus or manual. 

<button data-tooltip="Tooltip on Click" data-tooltip-trigger="click">Click Me!</button>

<button data-tooltip="Tooltip on Hover" data-tooltip-trigger="hover">Hover Me!</button>

<input data-tooltip="Tooltip on Focus" data-tooltip-trigger="focus" value="Focus on Me!">

<button id="manual-tooltip" data-tooltip="Manual Tooltip" data-tooltip-trigger="manual">Click button bellow</button>
